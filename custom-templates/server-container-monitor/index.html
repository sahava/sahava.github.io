Server Container Monitor - Custom Tag Template

Latest release: March 26, 2021 | Comments

The Server Container Monitor custom tag template is a Server container tag template for Google Tag Manager’s community template gallery.

Resource
Blog post
Gallery entry
GitHub repo

Description

Use this template to set up a monitoring system for Google Tag Manager’s Server container tags. With the monitor, you can collect data about tags that fired (or did not fire) with any given container instance. This information can be used to proactively fix issues with your tags.

Instructions

The tag itself is simple to setup. You add the BigQuery table details (you need to just create the table first per the blog post).

You’ll need to configure all your tags to pass the correct metadata. Follow the links above to the blog post to learn how to do this.
