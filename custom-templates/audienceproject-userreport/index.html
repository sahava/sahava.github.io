    AudienceProject UserReport - Custom Tag Template

    Latest release: November 5, 2020 | Comments

    The AudienceProject UserReport custom tag template is an unofficial tag template for Google Tag Manager’s community template gallery.

    Resource
    Vendor documentation
    Gallery entry
    GitHub repo

    Description

    This template loads the UserReport SDK on the site, and it also initializes the global _urq queue for passing section pageviews and other items.

    You can use the template to track section views by choosing the respective option from the drop-down menu, and then adding the Section ID (either directly or via a Google Tag Manager variable) to the field that appears.
