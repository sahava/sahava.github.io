Enhanced Ecommerce Object Builder - Custom Variable Template

Latest release: September 16, 2019 | Comments

The Enhanced Ecommerce Object Builder custom variable template is a variable template for Google Tag Manager’s community template gallery.

Resource
Blog post
Gallery entry
GitHub repo

Description

The Enhanced Ecommerce Object Builder template can be used to build a fully functional Enhanced Ecommerce object.

When you build a variable with this template, you populate the fields with whatever hard-coded values or Google Tag Manager vairables you consider important. Then, you add the variable to the “Ecommerce” settings of your Google Analytics tag (or, in some cases, your Google Analytics Settings variable).

The variable is added by unchecking “Use Data Layer”, and choosing the variable from the drop-down list that appears.

You can find more detailed instructions in the blog post.
